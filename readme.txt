This plugin generates product content, product images, product tags, meta tags and meta description for WooCommerce products using AI.





== Description ==





Webbersai Product Content Generator For Woocommerce is a plugin that automatically generates content, product images, meta tags, and description for your products using AI technology. This helps you to quickly and easily create SEO-friendly content for your product pages.





= Features =





* Generate product descriptions and short descriptions.


* Generate SEO meta tags and meta descriptions.


* Generate product tags.


* Generate product images.


* Works seamlessly with TinyMCE editor.


* Easy to use and configure.





== Installation ==





1. Upload the plugin files to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.


2. Activate the plugin through the 'Plugins' screen in WordPress.


3. Navigate to the product edit screen to start generating content.





== Admin Settings Configure ==





1. Select Type of AI.


2. Add your API key.


3. Add your Organisation model for OpenAI.


4. Select your AI model.


5. Select your content language.





== Frequently Asked Questions ==





= How do I generate content for my products? =





Navigate to the product edit screen, enter the product title, and click the "Generate Product Description" button. It will automatically generate product description, short description, and tags. Then click the "Generate Meta Tags" button. It will automatically generate meta tags and description for your product.
